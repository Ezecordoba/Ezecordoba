dinámicas y KPI a partir de datos empresariales en SQL Server.  Implementé un proceso ETL automatizado en Python, desarrollé el backend con FastAPI y el frontend con React. La solución fue desplegada en la intranet de la empresa mediante Windows IIS, con autenticación de usuarios y backend ejecutado como servicio mediante NSSM.
